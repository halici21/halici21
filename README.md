## Featured Project

### Lunar Orbit Determination Framework

A Python research framework and PyQt5 desktop application for comparing orbit
determination estimators in lunar tracking scenarios.

Key capabilities:

- Moon-centered trajectory propagation
- Earth and Sun third-body perturbations
- Ground-station visibility and lunar occultation
- Synthetic range, angle, range-rate, and two-way counted Doppler measurements
- BLS-LM, SRIF, and SR-UKF estimator workflows
- Monte Carlo, fragmented-visibility, and long-duration campaign scripts
- Diagnostics for residuals, consistency, observability, covariance condition,
  and runtime

Repository:
[lunar-orbit-determination](https://github.com/halici21/lunar-orbit-determination)
